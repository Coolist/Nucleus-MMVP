Tasks

Easily create tasks that let your smart devices talk to each other. Create tasks such as turning off the lights when you’re away or lock the doors and turning down the thermostat when you go to bed.

Trigger Examples

A trigger is what activates your task. You can select one or multiple triggers.

Types of triggers include: Time, Date, Weather, Devices, Sensors, or Location

If it is raining after 9:00 AM
If the thermostat is set to away on Saturday
If I arrive home before 6:00 PM
If the front door is unlocked

Action Examples

An action is what your task does when triggered. You can combine multiple actions together.

Types of actions include: Changing device states (lights, locks, switches), or Notifications

Set the light's colour to blue
Lock the front door
Set the thermostat to 68°
Turn on the living room lamp
